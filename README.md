# Проект: Путешествие по России

**Самые интересные места для вашего отпуска**

* Флексбокс верстка
* Позиционирование элементов
* Псевдоклассы
* Псевдоэлементы
* Плавность переходов
* Гриды
* Разработка интерфейса для разных устройств
* Медиазапросы
